Banana Republic Online

Transitioning from print to web to responsive design mindset takes a fair amount of creative shifting. This project involved working alongside the Banana Republic web design team to develop a fresh interactive, responsive online experience.

process

Following what can be considered a Lean UX model, a series of worksessions were held with the client to iteratively develop user personas, web brand pillars and experience guidelines.

The initial draft of the Prototype was created in a collaborative prototyping tool named Axure. Although very powerful, we discovered that the program does not handle responsive design well and, as a result, I switched to developing a fully responsive HTML/CSS prototype.

Final thoughts

Redesigning a website for an established brand affiliated with a large corporation is a very creative experience. I enjoyed immersing myself in a design environment and discovering new ways to innovate within the constraints of the project.
